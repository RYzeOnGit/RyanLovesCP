# Common Algorithms for Division 2

This folder contains implementations of algorithms commonly needed in Division 2 contests.

## Files

- `kattis_io_helper.py` - Standard I/O patterns for Kattis
- `prefix_sums.py` - Prefix sum arrays for range queries
- `binary_search_variants.py` - Different binary search patterns

## Quick Reference

### When to Use What

**Prefix Sums**
- Range sum queries
- Subarray problems
- Cumulative problems

**Binary Search**
- Sorted array search
- Optimization problems ("find maximum/minimum such that...")
- Answer space search

**Graph Algorithms** (see templates/)
- BFS: Shortest path in unweighted graph
- DFS: Connectivity, cycles, traversal
- Union-Find: Connected components

**Dynamic Programming** (see templates/)
- Optimization problems with overlapping subproblems
- Counting problems
- Grid problems

**Greedy** (see solved_examples/)
- Optimization problems with local choices
- Scheduling problems
- Interval problems
