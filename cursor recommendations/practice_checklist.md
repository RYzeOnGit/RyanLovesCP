# UAPC 2026 Division 2 - Practice Checklist

## Pre-Contest Practice Plan

### Week 1-2: Fundamentals
- [ ] Master I/O patterns (Kattis format)
- [ ] Practice array manipulation problems
- [ ] Solve 10+ string problems
- [ ] Learn basic sorting and searching

### Week 3-4: Algorithms
- [ ] Master BFS and DFS
- [ ] Practice greedy algorithms
- [ ] Learn basic dynamic programming
- [ ] Solve graph connectivity problems

### Week 5-6: Advanced Topics
- [ ] Binary search variants
- [ ] Union-Find problems
- [ ] Prefix sums and range queries
- [ ] Math problems (GCD, LCM, primes)

### Week 7-8: Contest Practice
- [ ] Solve UAPC 2022 Division 2 problems
- [ ] Solve UAPC 2023 Division 2 problems
- [ ] Solve UAPC 2024 Division 2 problems
- [ ] Solve UAPC 2025 Division 2 problems
- [ ] Time yourself (aim for 30-60 min per problem)

## Problem Types to Master

### Easy (Must Solve)
- [ ] Simple I/O and arithmetic
- [ ] Array manipulation
- [ ] String basics (palindromes, anagrams)
- [ ] Basic sorting
- [ ] Simple greedy

### Medium (Should Solve)
- [ ] Graph traversal (BFS/DFS)
- [ ] Dynamic programming basics
- [ ] Binary search
- [ ] Two pointers
- [ ] Sliding window

### Hard (Nice to Solve)
- [ ] Advanced DP
- [ ] Graph algorithms (shortest paths)
- [ ] Union-Find
- [ ] Math problems
- [ ] Complex greedy

## Daily Practice Routine

### 30 Minutes
1. Read 2-3 problems (don't solve, just read)
2. Identify problem types
3. Think about approach

### 1 Hour
1. Solve 1-2 easy problems
2. Review solution
3. Note patterns

### 2 Hours
1. Solve 1 medium problem
2. Debug if needed
3. Compare with optimal solution
